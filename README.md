# Spaceborne River Hydrometry

This repository provides the core code and representative workflows developed for the manuscript:

**Capturing Hidden Rivers with Spaceborne River Network Hydrometry: Discharge Retrieval in Intermittent Rivers**

The repository currently includes the principal modules required to understand and reproduce the main methodological workflow, including:

* multisource remote-sensing data preprocessing;
* intermittent river-surface extraction;
* river-surface connectivity refinement;
* multi-temporal river-width retrieval;
* soil-moisture-constrained discharge estimation;
* model evaluation and visualization.

This repository represents the review-stage release of the project. To ensure code clarity, data security, and compliance with third-party data-use restrictions, selected operational scripts, licensed datasets, large intermediate products, and region-specific processing utilities are not included in the present version. The repository will be further organized and expanded following manuscript acceptance.

For questions regarding the code or reproducibility, please contact the corresponding author.
